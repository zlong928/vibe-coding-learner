# Vibe Coding Learner - 项目规划

## 项目概述

- **项目名称**: vibe-coding-learner
- **目标**: 从零开始学习 Vibe Coding（AI辅助开发），掌握开发者必备技能，具备代码审查能力
- **学习方式**: 脱产学习，每天8小时，边学边问
- **总时长**: 1周（40小时）
- **记录工具**: Obsidian

## 用户画像

| 属性 | 值 |
|------|-----|
| 编程基础 | 0基础 |
| 学习速度 | 很快 |
| 学习风格 | 实践导向，边做边学 |
| 主动性 | 高，会主动提问 |

## 核心目标

1. **掌握开发环境** - VS Code + Cursor/Windsurf + Git
2. **编程入门** - HTML/CSS/JavaScript 基础
3. **AI 辅助开发** - 使用 AI 工具进行开发
4. **代码审查能力** - 理解代码审查流程和最佳实践

## 关键约束

- 时间紧凑：仅1周
- 0基础：从零开始
- 实践导向：边做项目边学
- 记录优先：使用 Obsidian 记录学习笔记
