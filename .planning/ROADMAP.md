# ROADMAP - Vibe Coding Learner

**Total Duration**: 1 week (40 hours)  
**Learning Mode**: Intensive, 8 hours/day, hands-on

---

## Phase 1: 开发环境搭建 (Day 1)

**目标**: 配置完整的开发环境

**Requirements:**
- ENV-01: 安装并配置 VS Code
- ENV-02: 安装并配置 Cursor/Windsurf AI IDE
- ENV-03: 安装并配置 Git 版本控制
- ENV-04: 掌握终端基础命令

**交付物:**
- 配置好的开发环境
- GitHub 账号和仓库
- 第一个 Git 提交

---

## Phase 2: 编程基础 - HTML/CSS (Day 2)

**目标**: 掌握网页结构与样式

**Requirements:**
- HTML-01: 理解 HTML 文档结构
- HTML-02: 常用 HTML 标签使用
- CSS-01: CSS 选择器和盒模型
- CSS-02: 布局基础 (Flexbox/Grid)

**交付物:**
- 个人简历网页

---

## Phase 3: 编程基础 - JavaScript (Day 3-4)

**目标**: 掌握 JavaScript 核心概念

**Requirements:**
- JS-01: 变量、数据类型、运算符
- JS-02: 函数和作用域
- JS-03: DOM 操作
- JS-04: 事件处理
- JS-05: 异步基础 (Promise/async-await)

**交付物:**
- 交互式待办事项应用

---

## Phase 4: Vibe Coding 专精 (Day 5)

**目标**: 掌握 AI 辅助开发流程

**Requirements:**
- VIBE-01: Prompt Engineering 基础
- VIBE-02: 使用 Cursor/Windsurf 进行开发
- VIBE-03: AI 代码审查与优化
- VIBE-04: 调试与排错技巧

**交付物:**
- AI 辅助开发的简单项目

---

## Phase 5: 代码审查能力 (Day 6)

**目标**: 理解代码审查流程

**Requirements:**
- REVIEW-01: 代码审查的目的和流程
- REVIEW-02: 常见代码问题识别
- REVIEW-03: 代码最佳实践
- REVIEW-04: Pull Request 流程

**交付物:**
- 代码审查报告

---

## Phase 6: 实战项目 (Day 7)

**目标**: 综合运用所有技能

**Requirements:**
- PROJECT-01: 项目规划与设计
- PROJECT-02: 完整项目开发
- PROJECT-03: 代码审查与优化
- PROJECT-04: 项目演示与总结

**交付物:**
- 完整的个人项目

---

## 学习节奏

| Day | 主题 | 重点 |
|-----|------|------|
| 1 | 环境搭建 | 工具配置 |
| 2 | HTML/CSS | 网页结构与样式 |
| 3-4 | JavaScript | 编程逻辑 |
| 5 | Vibe Coding | AI 辅助开发 |
| 6 | 代码审查 | 审查流程 |
| 7 | 实战 | 综合项目 |
